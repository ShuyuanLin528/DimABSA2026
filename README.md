# Dimensional Aspect-Based Sentiment Analysis Capstone Project

## Project Overview
This repository contains my undergraduate final-year thesis project on Dimensional Aspect-Based Sentiment Analysis (DimABSA). The project focuses on multilingual aspect-level sentiment analysis, including structured extraction and continuous、sentiment prediction.

## Why this project matters
Traditional sentiment analysis often predicts only coarse labels such as positive or negative. In this project, I explore a more fine-grained setting where the system identifies aspect-opinion structures and predicts continuous affective values such as valence and arousal.

## My contribution
In this project, I:
- studied recent research on multilingual and aspect-based sentiment analysis
- designed and implemented experimental pipelines for DimABSA subtasks
- conducted multilingual experiments across different datasets and settings
- analyzed model errors and compared different methodological choices
